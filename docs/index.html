The Tintri APIs provide a way to manage Tintri's virtual aware storage. APIs provide a developer tools to create applications and automation workflows. Currently, only HTTP REST APIs are supported.

Functionality includes obtaining information on:

  • appliance
  • alerts
  • performance statistics
  • snapshots
  • virtual machines
  • virtual disks
  • datastores
  • licensing
  • RBAC
  • replication
  • service groups

As well as obtaining information, the APIs control the following:

  • appliance
  • alert reports
  • autosupports
  • datastore reports
  • licensing
  • service groups
  • RBAC
  • replication
  • snapshots
  • virtual machine reports

HTTP

This HTTP interface is RESTful with JSON payloads. The download includes documentation and samples.
