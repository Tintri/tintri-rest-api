Method Summary
ResourceDescription
GET /v310/appliance/{uuid}/certificateReturns any present host certificate and trusted Certificate Authority (CA) certificates.
PUT /v310/appliance/{uuid}/certificateUpload a certificate to VMstore. The uploaded Certificate object must contain these attributes: - type (SIGNED or CA), and - content (certificate in PEM format). For Signed certificate, the content must contain both the certificate and the key, in that order. For CA certificate, the content must contains only the certificate. Chained certificates may be appended. The certificates are only used for LDAP authentication when TLS is enabled.
