Path: / v310 / servicegroup / syncRepl / resume

Method Summary
ResourceDescription
POST /v310/servicegroup/syncRepl/resumeOn VMStore, this will resume all synchronous replication configurations at the system level. On TGC, this will resume all the synchronous replication configurations, that the VMstore identified by the specified uuid, participates in. It is recommended to send this request to both primary and secondary to avoid unwanted error messages.If some synchronous replication configurations fail to be resumed, we will continue to try to resume rest of the configurations. The return object will contain a list of TintriError where code contains the error code ("SyncReplErrNoError" means success), and targetId contains the synchronous replication service group ID, and targetName contains the synchronous replication service group name.
