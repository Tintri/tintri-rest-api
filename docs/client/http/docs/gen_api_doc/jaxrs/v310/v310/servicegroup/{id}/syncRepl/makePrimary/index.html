Path parameters:

Method Summary
ResourceDescription
POST /v310/servicegroup/{id}/syncRepl/makePrimaryInitiate a failover from this VMstore. This request can be sent to either primary or secondary VMstore or TGC. Both primary and secondary sides state must be OK. Otherwise, operation is rejected with specifc error code. The API initiates the failover operation with filesystem and continues to poll for the expected state transition. If expected state transition is reached within 30 seconds, it returns a success. Otherwise, it returns an error.
