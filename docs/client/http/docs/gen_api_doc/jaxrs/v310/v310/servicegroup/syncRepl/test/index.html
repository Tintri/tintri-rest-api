Method Summary
ResourceDescription
POST /v310/servicegroup/syncRepl/testTest the Synchronous Replication configuration associated with the specified . The synchronous replication configuration of this service group can not be null or error will be returned. Please note when this test is run, it will only test the connection between the primary VMstore and the secondary VMstore as given in the configuration e.g. VMstore A as primary and VMstore B as secondary. It will not test the connection after potential failover, e.g. B as primary and A as secondary. If users want to test the fail over scenario they will need to call this API again with primary/secondary roles reversed. With this API, TGC will execute the test operation on all the participating VMStores of the synchronous replication configuration, and return the combined result. If an error is returned from one or more VMStores, then this error will be propagated to the response test result.
