Path: / v310 / appliance / {uuid} / escalateDPAPrivileges

Path parameters:
uuid -
Method Summary
ResourceDescription
PUT /v310/appliance/{uuid}/escalateDPAPrivilegesEscalates DATA_PROTECTION_ADMIN user's privilges until the time specific by escalatedPrivilegeExpiryTimeMs. If the time is in past, privileges aren't escalated.
With escalated privileges, a DATA_PROTECTION_ADMIN user can perform below operations:
  • Unlock snapshot
  • Remove lock feature from VM snapshot schedule
  • Remove lock feature from service group
  • Disable the Lock Snapshot feature
  • Resume snapshot deletion
