uuid - The UUID of the datastore.
Method Summary
ResourceDescription
GET /v310/datastore/{uuid}/replicationConfigReturns the system-wide replication configuration
PUT /v310/datastore/{uuid}/replicationConfigUpdate the system replication configuration, for example, from system disabled to system enabled. While applying the system default replication configuration to the VMs, the VMs with a custom outgoing replication configuration will be ignored. In addition to this, a VM which has an incoming replication configuration, or is a part of service group or is not live and does not have any snapshot or has snapshots but its latest snapshot is a replica, will be marked as ineligible for system default configuration, and a disabled system default replication configuration will be added for such a VM instead.
