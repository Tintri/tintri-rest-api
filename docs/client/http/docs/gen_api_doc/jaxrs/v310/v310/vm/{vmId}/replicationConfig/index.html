Method Summary
ResourceDescription
DELETE /v310/vm/{vmId}/replicationConfigDeprecated. Delete all the incoming and custom outgoing replication configurations for a VM. Deprecated since v310.21. Please use or DELETE APIs for a similar operation. System default replication configuration will not be deleted by this operation. After successfully deleting the custom replication configurations, an attempt will be made to automatically pickup or add the system default outgoing replication configuration for this VM, if one has not been added already.
POST /v310/vm/{vmId}/replicationConfigDeprecated. Create a new replication configuration for the VM. Deprecated since v310.21. Please use or APIs for similar operations. This API will first cleanup the stale or existing system default replication configuration for the specified VM if one exists, and then try to create a new outgoing replication configuration as specified, for this VM. If a valid configuration already exists for the VM, then the new or specified configuration is ignored.
PUT /v310/vm/{vmId}/replicationConfigUpdate the passed replication (repl) configuration.
