Path: / v310 / vm / testRelocateVms

Method Summary
ResourceDescription
PUT /v310/vm/testRelocateVmsValidate the relocate VMs request for whether the provided VMs can be migrated to the specified destination datastore. Validation will check for whether the source and destination datastore have proper license and that the VMs meet the eligibility requirements such as the VMs cannot have high frequency snapshot protection, have disks on multiple datastores, SRM enabled, quota enabled, and synchronous replication enabled.
