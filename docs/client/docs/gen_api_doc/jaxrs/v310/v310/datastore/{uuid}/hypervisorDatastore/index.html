Method Summary
ResourceDescription
GET /v310/datastore/{uuid}/hypervisorDatastoreGet all hypervisor datastores.

Supports vmId as an optional query parameter to filter the hypervisor datastore by VM UUID.

Please note when the vmId represents a synchronous replication VM, this API only returns the datastore that is hosted on that synchronous replication submount.
