Path: / v310 / vm / replicationConfig

Method Summary
ResourceDescription
POST /v310/vm/replicationConfigDeprecated. Create a new replication configuration for the specified VMs. Deprecated since v310.21. Please use or POST APIs for a similar operation. This API will first cleanup the stale or existing system default replication configuration for the specified VMs, if one exists, and then try to create a new outgoing replication configuration as specified, for each VM. If a valid VM replication configuration already exists then the new or specified configuration is ignored for that VM.
