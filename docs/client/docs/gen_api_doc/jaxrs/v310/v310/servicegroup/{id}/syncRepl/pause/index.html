Path parameters:
id - Service Group ID
Method Summary
ResourceDescription
POST /v310/servicegroup/{id}/syncRepl/pausePause the specified synchronous replication configuration. This API will temporarily stop synchronous replication from primary to secondary so that NFS client only writes to primary. This request can be sent to either primary or secondary. If the service group is invalid, RESOURCE_NOT_FOUND error will be returned. Pause a already paused configuration will return successfully with no operation. TGC will first update the status of the synchronous replication configuration locally and then make attempts to pause the synchronous replication config on all the participating VMStores. If the call fails on one or more VMStores, then a policy error will be generated and TGC will retry executing these calls until success.
