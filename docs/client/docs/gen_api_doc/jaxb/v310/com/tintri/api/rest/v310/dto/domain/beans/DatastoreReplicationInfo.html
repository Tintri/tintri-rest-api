Attributes
NameTypeDescription
acceptIncomingTrafficxsd:booleanWhen this datastore acts as the destination of asynchronous and synchronous replication, the flag indicates whether the incoming traffic is accepted or not. When set to false, all asynchronous and synchronous replications coming into this target are stopped. When set to true, all asynchronous and synchronous replications coming into this target will resume.
acceptIncomingTrafficCloudxsd:booleanWhen this datastore acts as destination of a cloud replication, the flag indicates whether the incoming traffic is accepted or not. When set to false, all cloud replications coming into this target are stopped. When set to true, all cloud replications coming into this target will resume.
allowOutgoingTrafficxsd:booleanWhen this datastore acts as the source of asynchronous and synchronous replication configuration this flag indicates whether asynchronous and synchronous replication is enabled or not. When set to false, all asynchronous and synchronous replications originating from this source are stopped. When set to true, all asynchronous and synchronous replications originating from this source will resume.
allowOutgoingTrafficCloudxsd:booleanWhen this datastore acts as the source of cloud replication configuration this flag indicates whether cloud replication is enabled or not. When set to false, all cloud replications originating from this source are stopped. When set to true, all cloud replications originating from this source will resume.
