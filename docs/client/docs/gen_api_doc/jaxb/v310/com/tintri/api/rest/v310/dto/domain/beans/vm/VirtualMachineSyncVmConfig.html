Name: VirtualMachineSyncVmConfig

Properties of virtual machine that represents the auto detach virtual disk functionality. When 'isDiskAutoDetach' is true, Tintri OS is responsible to automatically remove the virtual disk from virtual machine after the specify 'diskAutoDetachInMinutes' period.

Data Structure
{
   "typeId": "com.tintri.api.rest.v310.dto.domain.beans.vm.VirtualMachineSyncVmConfig",
   "diskAutoDetachInMinutes": <Number>,
   "diskWhenAdded": DateTime,
   "isDiskAutoDetach": <Boolean>,
}
Attributes
NameTypeDescription
diskAutoDetachInMinutesxsd:intIndicates in minutes when the virtual disk will be automatically removed from the virtual machine. Default value is 2880 minutes (48 hours).
diskWhenAddedDateTimeObtains the time when SyncVM File-Level Restore disk is added to virtual machine.
isDiskAutoDetachxsd:booleanChecks if Tintri OS auto detach virtual disks. Default is true.
