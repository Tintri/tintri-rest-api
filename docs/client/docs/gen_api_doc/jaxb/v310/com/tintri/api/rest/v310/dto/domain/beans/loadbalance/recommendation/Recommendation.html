Name: Recommendation

Describes the conditions of a VMstore Pool based on the analysis performed by the Recommendation Engine. The Recommendation contains some of the following concepts:
Description Property
current issues in the Recommendation issues
actions that can be done to alleviate or mitigate the issues actionGroups
improvements expected if the recommendation is accepted expectedOutcomes
state of the recommendation including the progress applying the recommendation state
Tintri Global Center will generate the Recommendation for a VMstore pool on a regular schedule, currently 6 am and 6 pm local time. The Recommendation is generated even if there are no issues detected.
